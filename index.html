Paste a public GitHub repo. Get a DeSlop score.

The browser pass fetches supported public files and returns concrete findings with file, line, matched text, reason, and fix direction.

npx -y github:TheArtOfSound/DeSlop -- . --json

Analyze public repo

Paste a public GitHub repository URL. Browser mode scans up to 80 supported files under 250 KB each.

Ready.

Inputs

Markdown, text, JSON, HTML, CSS, JavaScript, TypeScript, JSX, and TSX files from a public GitHub repo.

Outputs

Score, severity counts, exact match, file, line, reason, and fix direction.

Non-goal

DeSlop does not decide whether code was written by a person or a model.

Report contract

{
  "schemaVersion": 1,
  "summary": {
    "score": 100,
    "filesScanned": 15,
    "findingsTotal": 0
  },
  "findings": []
}
