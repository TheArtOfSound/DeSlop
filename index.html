DeSlop
Run Scope Report Verify GitHub
Command line repo audit

Find vague copy and fake-complete residue before review.

DeSlop checks product copy and source files for concrete evidence: weak errors, dead navigation, debug output, browser-only trust state, unfinished branches, and claims without evidence.

Repository Deploy runs Commands
Run
npm install
npm run audit -- .
npm run audit -- . --json
npm run audit -- . --fail-on high
npm run audit -- . --min-score 90
npm run audit -- . --max-file-bytes 250000
npm run verify:page

Inputs

Markdown, text, JSON, HTML, CSS, JavaScript, TypeScript, JSX, and TSX files under the scan path.

Outputs

Text or JSON reports with score, category counts, exact match, line, reason, and fix direction.

Non-goal

DeSlop does not decide whether code was written by a person or a model. It reports visible repo evidence.

Report contract

{
  "schemaVersion": 1,
  "summary": {
    "score": 100,
    "filesScanned": 15,
    "findingsTotal": 0
  },
  "findings": []
}
