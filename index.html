DeSlop
Run Scope Report Verify GitHub

Repo audits for software that feels unwired.

DeSlop scans product copy and source files for concrete signals: vague wording, weak errors, dead navigation, debug residue, browser-only trust state, and claims without evidence.

View repo Read commands Deploy runs 
npm install
npm run audit -- .
npm run audit -- . --json
npm run audit -- . --fail-on high
npm run audit -- . --min-score 90
npm run audit -- . --max-file-bytes 250000

What it checks

Copy, UX, implementation, security, and release hygiene findings with file, line, matched text, reason, and fix direction.

What it avoids

It does not claim who wrote the code. It scores visible repo evidence and keeps findings specific.

Why it exists

Repos can look finished before the core behavior is wired. DeSlop catches that residue earlier.
